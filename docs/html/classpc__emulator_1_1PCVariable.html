PCVariable* pc_emulator::PCVariable::GetPtrToField ( string  NestedFieldName)

Creates a new variable which points to a sub field.

The memory location of the newly created variable points to the location where the content of the nested field was stored in the original variable.

See also
__AccessedFields A new entry is added to the hashmap. If the same subfield is requested again, a new variable is not created. Instead the previously created variable is returned again.
Parameters
NestedFieldNameThe Byte and Bit Offsets and MemLocation of the new variable are set to match the physical location where this nested field is stored
